# Forever — AI Proposal Planner

> Plan the moment that changes everything.

AI-powered romantic planning assistant built for the **AMD Developer Hackathon: ACT II**.

Every word is generated by **Kimi K2** — an open AI model — running on **AMD hardware** through the **Fireworks AI API**.

---

## Features

- **Proposal Planner** — Speech, venue recommendations, timing, ring advice, and a full checklist
- **Vow Writer** — Personalized wedding vows in any tone
- **Love Letter** — Heartfelt letters for any occasion
- **Ring Advisor** — Ring recommendations based on personality, lifestyle, and budget
- **Date Ideas** — Five creative date ideas tailored to your style

## Tech Stack

- **Frontend:** React 19, TypeScript, Vite, Tailwind CSS v4, Porsche Design System
- **AI Model:** Kimi K2 via Fireworks AI API
- **Hardware:** AMD GPUs (via Fireworks AI)
- **Routing:** React Router v7

## Setup

### 1. Install dependencies

```bash
npm install
```

### 2. Add your Fireworks AI API key

Create a `.env` file in the project root:

```
VITE_FIREWORKS_API_KEY=your_fireworks_api_key_here
```

### 3. Run locally

```bash
npm run dev
```

Open http://localhost:5173

### 4. Build for production

```bash
npm run build
```

## Docker

```bash
# Build
docker build --build-arg VITE_FIREWORKS_API_KEY=your_key -t forever .

# Run
docker run -p 3000:3000 forever
```

Open http://localhost:3000
